# 🜏 oliver-audit-agent

**GCP Cost Audit Agent** — Your demonic sentinel that rends the veils of cloud spending.

This Node.js-powered agent continuously monitors Google Cloud Platform billing exports, breaks down costs by service/SKU/project, detects anomalies, and channels structured insights directly into **Looker Studio** (or any BI tool) and custom **interactive HTML dashboards**.

Built for the lazy-yet-powerful sorcerer, it supports multiple AI engines (Gemini, OpenAI, Claude) for intelligent summaries and recommendations.

## 🔥 Core Powers
- Real-time GCP billing analysis via BigQuery
- AI-generated operational reports & actionable recommendations
- Automatic incident logging into a dedicated dashboard table
- Mock/demo mode for safe development and presentations
- Multi-AI provider routing (switch instantly)
- Ready for Looker Studio + custom Windows BI-style HTML exports

## 666 Why It Exists
Tired of surprise GCP bills rising from the abyss? This agent stands eternal watch, alerting you before costs spiral into the void.

Perfect for developers, SREs, and finance teams running production workloads on Google Cloud.
